# Heart Disease Prediction using Machine Learning

## Overview

This project predicts the presence of heart disease using Machine Learning techniques.
The model analyzes various medical attributes such as age, cholesterol level, chest pain type, and heart rate to determine whether a patient is likely to have heart disease.

The implementation is done using **Python in a Jupyter Notebook**.


## Dataset

The dataset used in this project contains medical information of patients and a target variable indicating the presence of heart disease.

### Features in the dataset include:

* Age
* Sex
* Chest Pain Type
* Resting Blood Pressure
* Cholesterol
* Fasting Blood Sugar
* Resting ECG
* Maximum Heart Rate
* Exercise Induced Angina
* ST Depression
* Slope of Peak Exercise ST Segment
* Number of Major Vessels
* Thalassemia

The dataset file used in this project is:

heart.csv


## Project Files

Heart Disease Prediction.ipynb – Jupyter Notebook containing the complete machine learning workflow
heart.csv – Dataset used for training and testing the model
